02

Color

One blue ramp, one warm-neutral ramp, and four contrast-checked status colors. Reference tokens by name — never by hex value.

The single most important color rule: The signature brand blue #0BA1E2 (--wzjh-blue-500) is gorgeous as a fill, stroke, or logo — but at 2.91:1 contrast on white it is invisible-grade for text. Any link, button label, or UI text in brand blue is a defect. Use var(--wzjh-action) (#0A6FB0, 5.37:1) for all interactive elements and text.

05

Voice & Tone

Authoritative. Precise. Plain. Lead with verifiable performance data — never with adjectives or exclamation marks.

Authoritative

State facts. Quote standards. Cite measurements. "Engineered for 1,000,000 mechanical operations" — not "incredibly durable."

Precise

Use exact values, correct units, and defined abbreviations. Ambiguity in spec copy has direct operational consequences on the factory floor.

Plain

No hype. No exclamation marks. No superlatives without supporting data. Write for an engineer who trusts evidence over enthusiasm.

Unit & notation rules

No spaces between value and unit. Exact casing. Use Ø for diameter, not O or 0.

IP67 IK10 24V 380V AC/DC 10A Ø22mm −25°C…+70°C 1NO+1NC 1,000,000 ops

07

Building with AI tools

Any AI tool — Claude, ChatGPT, Gemini, Cursor, v0, Copilot — can generate perfectly on-brand JINHONG pages and presentations. Follow four steps.

01

Load the spec

Paste wzjh-design-system.md as context. This is the single source of truth: colors, type, spacing, components, and voice in one document.

02

Add the guardrails

Also paste FOR-AI.md. It contains the strict generation rules and the 10-point self-check the AI must run before returning any output.

03

Link the stylesheets

Link tokens/wzjh-tokens.css then wzjh.css — always in that order. Use .wzjh-* classes and var(--wzjh-*) tokens only. No hardcoded hex values.

04

Run the self-check

Before accepting any output, run the 10-point self-check in FOR-AI.md §9. Every box must be checked. Reject and fix any unchecked item.

Token machine format: For Figma, Style Dictionary, or programmatic tooling, use tokens/wzjh-tokens.json — it mirrors wzjh-tokens.css 1:1 in W3C DTCG format. For slide decks, add FOR-PRESENTATIONS.md as additional context.
